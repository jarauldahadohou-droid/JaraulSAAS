# IlimiNetZone

## Vision

IlimiNetZone est une plateforme SaaS permettant à des utilisateurs d'acheter un accès Internet sécurisé via une application mobile.

Le service permet :

- d'acheter un forfait Internet ;
- d'effectuer le paiement via Mobile Money ;
- d'activer automatiquement un accès réseau sécurisé ;
- de gérer son abonnement depuis une application mobile.

## Mission

Rendre l'accès à Internet simple, rapide et accessible grâce à une expérience utilisateur moderne.

## Objectifs V1

- Application Android
- Préparation iOS
- Paiement Mobile Money
- Tableau de bord administrateur
- Gestion des abonnements
- Activation automatique des accès
- Connexion sécurisée

## Public cible

Particuliers situés au Bénin souhaitant acheter un accès Internet depuis une application mobile.

## Valeurs

- Simplicité
- Fiabilité
- Sécurité
- Rapidité
- Transparence
